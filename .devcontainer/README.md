# Dev Container Konfiguration

Dieser Dev Container verwendet das offizielle Home Assistant Dev Container Image für die Add-On-Entwicklung.

## Verwendetes Image

- `ghcr.io/home-assistant/devcontainer:2-addons`

Dieses Image enthält alle notwendigen Tools für die Entwicklung von Home Assistant Add-ons.

## Container-Neuaufbau

Um den Container neu zu bauen:
1. Command Palette öffnen: `Strg+Shift+P` (Windows/Linux) oder `Cmd+Shift+P` (Mac)
2. Befehl auswählen: `Dev Containers: Rebuild Container`
